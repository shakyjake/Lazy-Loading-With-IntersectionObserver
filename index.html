No Frills Lazy Loading

To use, include the lazyloader.js file just before </body> (like this):

<script href="lazyloader.0.1.js"></script>

Then wrap your images in a noscript element with a class of nojs-image (like so):

<noscript class="nojs-image"> <img src="https://picsum.photos/990/557?image=456" alt="Image 456" /> </noscript>

This also works for picture elements or if your image is wrapped in an anchor

Please note: the noscript tag must only have 1 child.

Browser Support

The plugin works wherever IntersectionObserver is supported. For browsers that do not support it, there is a non-lazy fallback, provided querySelectorAll is supported (so IE7 and lower won't see any images).
